# 📱 Google Play Store Analytics

## 📌 Project Overview

Google Play Store Analytics is an end-to-end Data Analytics project that explores thousands of Android applications to uncover insights about app categories, ratings, installs, pricing, and user engagement.

The project demonstrates a complete analytics workflow including data cleaning, exploratory data analysis (EDA), business analysis, SQL querying, and data visualization using Python.

---

# 🎯 Business Problem

The Google Play Store contains thousands of applications across multiple categories. Understanding app performance can help businesses and developers identify market trends and improve decision-making.

This project answers questions such as:

- Which app categories are the most popular?
- Which categories receive the highest ratings?
- Which apps have the highest installs?
- What is the distribution of free vs paid apps?
- Which categories generate the highest user engagement?
- What is the average price of paid applications?
- Which applications receive the most reviews?

---

# 📂 Dataset

**Dataset:** Google Play Store Apps Dataset (Kaggle)

### Dataset Information

- Total Records: **10,841**
- Features: **13**
- Cleaned Records: **8,892**

# 💡 Business Insights

- Free applications dominate the Play Store, representing over **93%** of all apps.
- **Game** apps generate the highest installs and user engagement.
- **Family** is the largest app category on the platform.
- **Events** has the highest average user rating.
- Paid applications represent a small portion of the marketplace with an average price of approximately **$14**.
- Applications with high installs also tend to receive significantly more user reviews.
